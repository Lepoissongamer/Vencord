## VenCode

Adds an improved VenCode viewer to Discord.

<img width="1502" height="995" alt="image" src="https://github.com/user-attachments/assets/e2e10a02-f602-4768-a29f-e13c11282c2e" />


### Features

- Removes Discord’s line limit to view large files directly inside Discord
- Monaco editor for viewing and editing files directly
- Insert, insert and send, save, and close actions
- Supports multiple languages

### Monaco Editor

Edit files directly inside Discord with Monaco and send the modified version instantly.
